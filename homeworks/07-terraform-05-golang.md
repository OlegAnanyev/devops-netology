## Задача 3. Написание кода. 
Цель этого задания закрепить знания о базовом синтаксисе языка. Можно использовать редактор кода 
на своем компьютере, либо использовать песочницу: [https://play.golang.org/](https://play.golang.org/).

1. Напишите программу для перевода метров в футы (1 фут = 0.3048 метр). Можно запросить исходные данные 
у пользователя, а можно статически задать в коде.

1. Напишите программу, которая найдет наименьший элемент в любом заданном списке, например:
    ```
    x := []int{48,96,86,68,57,82,63,70,37,34,83,27,19,97,9,17,}
    ```
1. Напишите программу, которая выводит числа от 1 до 100, которые делятся на 3. То есть `(3, 6, 9, …)`.

##1
---------------------------------
``` go
package main
import "fmt"

func main() {
	fmt.Print("Введите расстояние в метрах:")
	var metr float64
	fmt.Scanf("%f", &metr )
	metr = 2
 
	var foot float64
	foot = metr / 0.3048

	fmt.Println("\nВ футах это будет: ")
	fmt.Println(foot)  
}
```

```
Введите расстояние в метрах:
12
В футах это будет: 
39.37007874015748
```

##2
 ---------------------------------

``` go
package main
import "fmt"

func main() {
	x := []int{48,96,86,68,57,82,63,70,37,34,83,27,19,97,9,17,}
	min := x[0]
	for _, element := range x {
		if element < min {
			min = element
		}  
	} 
	fmt.Println("В массиве")
	fmt.Printf("%v\n",x)
	fmt.Println("\nнаименьшее значение: ")
	fmt.Println(min)
}
```

```
В массиве
[48 96 86 68 57 3 63 70 37 34 83 27 19 97 9 17]

наименьшее значение: 
3
```

package main
import "fmt"

func main() {
	var array []int
    for i := 1; i <= 100; i++ {
        if i%3 == 0 {
        	array = append(array, i)
        }
    }
    fmt.Println("Следующие числа делятся на 3 без остатка:")
	fmt.Printf("%v\n",array)
}
```

```
